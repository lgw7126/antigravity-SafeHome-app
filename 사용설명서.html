1. 안심 귀가 호출이란?

본 서비스는 부모님(시니어)의 위치 파악 및 택시 호출 결제에 따르는 디지털 진입 장벽을 완전히 극복하기 위해 설계된 대리 호출 브릿지 플랫폼입니다.
직접 복잡한 앱을 이용해 예약을 완료할 필요 없이, "버튼 터치 단 한 번"으로 자녀의 스마트폰이나 PC 관제 창에 위성 GPS 지도를 전송하여 자녀가 안전하게 택시를 대신 예약하고 결제할 수 있도록 도와줍니다.

2. 대상 및 역할 구분

  • 👵 부모님용 화면 부모용
    • 대형 무입력(Zero-Typing) 위치 전송 버튼.
    • 오작동 방지 및 명확한 시각 안내 피드백.
    • 위치 차단 시 대체 수단: GPS 차단 시 네트워크 기지국 기반 위치 전송과 자녀 비상 다이렉트 전화(tel 링크) 연결 화면 제공.
  • 🧑‍💻 자녀용 화면 자녀 관제용
    • 실시간 부모님 위치 지도 마킹 및 상세 주소(도로명/지번) 역지오코딩.
    • 브라우저 3화음 알림벨 소리 및 OS 데스크톱 알림 팝업.
    • 카카오 T 앱 자동 연동 링크, 카카오맵 길찾기 연동 링크 및 주소 복사 기능.

3. 최초 설정 및 사용 방법

⚙️ 카카오맵 및 무선 브릿지 연동 설정 (최초 1회)

  1. 서비스 우측 상단의 ⚙️ 톱니바퀴 아이콘을 클릭합니다.
  2. 자신이 발급받은 카카오맵 JavaScript API Key를 입력해 줍니다.
  3. 자녀에게 다이렉트 전화를 걸기 위한 자녀 연락처(예: 01012345678)를 입력합니다.
  4. 기기 간 무선 연동을 위해 고유한 브릿지 연동 토픽 ID를 설정합니다. (양쪽 기기가 동일한 토픽 ID여야 함)
  5. 부모님이 자주 가시는 목적지 정보(예: 이름 '집' / 주소 '서울시 마포구...')를 최대 5곳까지 입력하고 저장합니다.
💡 카카오 API 키가 없는 경우에도 테스트할 수 있나요?
네, 가능합니다! API 키가 없어도 무료 전역 주소 API(Nominatim)가 자동으로 동작하여 부모님의 한글 주소와 위도/경도가 자녀 대시보드에 정상적으로 전송되며, 지도만 텍스트 정보로 대체됩니다. 데모 듀얼 뷰를 열어 즉시 작동을 시험해 보세요.

👵 부모님 이용 흐름 (전화 통화 & 목적지 원터치)

  • 위치 정보만 보내기:
    1. 상단의 거대한 [📢 위치전송] 버튼을 누릅니다.
    2. 위치 수집이 끝나면 화면에 "위치 전송 완료!" 팝업이 나타나며, **[이전 화면으로]** 버튼을 터치해 다시 첫 화면으로 복귀합니다.
  • 목적지 선택하여 전송 (시나리오 B):
    1. 중앙에 노출된 5가지 목적지 단추(예: [🏠 집으로], [👵 복지관으로], [👧 딸래미집으로] 등) 중 하나를 직접 터치합니다.
    2. 위치와 도착지가 자녀 화면에 한 번에 전달됩니다.
  • 자녀에게 전화해서 위치 알려주기 (시나리오 A):
    1. 첫 화면 최하단에 상시 배치된 초록색 [📞 자녀에게 전화해서 위치를 알려주세요] 대형 버튼을 누릅니다.
    2. 자녀에게 전화를 걸어 출발지와 가고 싶은 상세 목적지를 목소리로 대화하며 택시 호출을 요청합니다.

🧑‍💻 자녀 이용 흐름

  1. 자녀용 화면을 열어둡니다. (또는 듀얼 데모 뷰 사용)
  2. 부모님이 요청을 보내면 자녀 화면에 경고 사운드(C-E-G 화음)가 재생되며 데스크톱 알림창이 나타납니다.
  3. 배차하기:
    • 부모님이 시나리오 A로 보냈다면: [📋 출발 복사]를 눌러 주소를 복사하고 전화를 받아 목적지를 파악한 후 호출합니다.
    • 부모님이 시나리오 B로 보냈다면: [📋 출발 복사] 및 [📋 도착 복사]를 각각 클릭하여 카카오 T에 출발지와 목적지 주소를 입력합니다.
  4. 원격 목적지 주소 설정 및 동기화:
    • 자녀 화면 우측의 [부모님 목적지 동기화 설정] 패널에서 5개의 목적지 주소를 입력하고 [변경사항 부모님께 동기화]를 누르면, 인터넷 망을 통하여 부모님의 기기 화면에 새로운 목적지 버튼이 실시간으로 자동 업데이트됩니다.
  5. [🗺️ 카카오맵] 버튼을 클릭하면, 부모님 위치(출발지)와 목적지(도착지)가 모두 지정된 카카오맵 길찾기 웹 화면이 자동으로 열려 전체 경로를 쉽게 확인하고 길안내를 받을 수 있습니다.
